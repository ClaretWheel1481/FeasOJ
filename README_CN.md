# FeasOJ

#### 如何运行

1. 克隆此库。
2. 安装 Docker。
3. 运行 `npm install -g pnpm` 安装 Pnpm。
4. 运行 `pnpm i` 安装依赖项。
5. 运行 `pnpm dev` 启动前端服务器。
6. 运行 `cd backend-src\src` 和 `go mod tidy` 下载依赖。
7. 运行 `go run main.go` 启动后端服务器。

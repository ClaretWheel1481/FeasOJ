# FeasOJ
### 项目简介
FeasOJ 是一个基于 Vue 和 Golang 的在线编程练习平台，旨在为用户提供一个方便、高效的学习和练习环境。
<br>
[FeasOJ-Backend](https://github.com/ClaretWheel1481/FeasOJ-Backend)

### 项目结构
```
FeasOJ
│ 
├─public
├─src
│  ├─assets
│  ├─components
│  ├─pages
│  ├─plugins
│  ├─routers
│  └─utils
```

### 环境
- Vue 3
- Pnpm

### 如何运行
1. 克隆此库。
2. 运行 `npm install -g pnpm` 安装 Pnpm。
3. 运行 `pnpm i` 安装依赖项。
4. 运行 `pnpm dev` 启动前端服务器。
